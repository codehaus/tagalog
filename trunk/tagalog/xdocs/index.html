Overview

Tagalog is a collection of libraries targetted at the problem of parsing XML files that have been written by humans. The heart of the collection is tagalog-core, which is a simple framework for parsing XML documents in Java. tagalog-core is a thin layer over the top of an event-based XML parser (either SAX or XPP in the core package, though tagalog can be interfaced to other parsers quite easily). Events are delivered to tags based on the namespace of the corresponding element in a manner that is styled after the similar mechanisms in JSP and Jelly.
