Introduction

Tagalog provides a simple framework for parsing XML documents in Java. It takes a stream of events from an event-based parser and delivers them to implementations of the Tag interface. Tags are grouped into TagLibrarys, and a tag library is responsible for implementing whatever behaviour is needed during the parse (for example, building objects from some data model). The namespace-driven selection of tag libraries and tags is styled after the similar mechanisms in JSP and Jelly.

Taglog is intended to fill the gap between event-based parsers and full-blown tree-based XML processing APIs. If

  • you have a data model that you want to read from an XML file,
  • and extracting the information from a stream of events seems like hard work,
  • but you don't want to pull the whole document into a DOM tree,
Tagalog might be the solution you're looking for.
